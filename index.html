💻 PowerShell Automation

5 automation scripts built to streamline Service Desk operations

📄 Onboard-Employee.ps1
Creates AD user account, assigns to correct OU, grants folder permissions and generates onboarding report automatically.
📄 Offboard-Employee.ps1
Disables AD account, removes all group memberships, moves to Disabled Users OU, resets password and generates offboarding report.
📄 Bulk-UserCreation.ps1
Reads from CSV file to create multiple AD users with email addresses, department assignment and OU placement in one automated run.
📄 Hardware-Diagnostic.ps1
Analyzes CPU usage, RAM availability and disk health. Clears temporary files and generates diagnostic report for Service Desk records.
📄 Network-Diagnostic.ps1
Tests gateway and DNS connectivity, flushes DNS cache, renews IP address and resets TCP/IP network stack with full diagnostic logging.

🔐 Security Investigation

Phishing email investigation using multi-tool approach

📧

Email Header Analysis

Analyzed phishing email headers using MXToolbox and PhishTool to identify authentication failures and suspicious routing.

  • SPF: FAIL identified
  • DKIM: FAIL identified
  • DMARC: FAIL identified
  • Spam score: 8.4/10
🌍

IP Investigation (Kali)

Traced phishing email origin to Lithuania using WHOIS, confirmed no reverse DNS and identified suspicious ISP abuse contact.

  • WHOIS: Lithuania origin
  • HOST: NXDOMAIN confirmed
  • NSLOOKUP: No DNS records
  • RIPE database analyzed
📊

Comparison Analysis

Side by side comparison of legitimate internal email vs phishing email headers documenting key indicators of compromise.

  • Legitimate vs phishing routing
  • Authentication comparison
  • IOCs documented
  • Formal report filed

🚨 Phishing Indicators Identified

❌ Sender domain (servercorp-helpdesk.net) mimicking legitimate company domain (servercorp.local)
❌ SPF, DKIM and DMARC authentication all failed — sender not authorized
❌ Originating IP (185.234.219.44) traced to Lithuania via RIPE database
❌ No reverse DNS record (NXDOMAIN) — legitimate mail servers always have reverse DNS
❌ ISP abuse contact using Gmail address — not a legitimate ISP
❌ High spam score of 8.4/10 flagged by MXToolbox
❌ Urgency language designed to panic user into immediate action
